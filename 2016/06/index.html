iA Writer, Mobile Markdown编辑器

发表于 2016-06-16

前言

在豌豆荚上逛的时候看到iA Writer的介绍，一眼就看到了Markdown鹤立鸡群

“哎呀，以后可以在手机上写写字”

然后Copy到电脑上进行一些修改就可以发布到博客上来了

虽然只是文本编辑而已，用手机自带的便签之类的就完全可以胜任了

但是自我感觉一个支持Markdown的编辑器给予我更大的动力去“写作”

嗯，那么随便说一说正在用来码字的这个安卓Markdown编辑器(iA Writer)

界面

白纸，黑字，简约，精致，典雅

没有华丽的特效，缭乱的色彩

唯有安静的让你专注于码字

突然发现风格和魅族的系统输入法很搭

大道至简，不外如是

功能

功能么，一言以蔽之，一个移动端Markdown编辑器

导出(Export)

支持以下几种格式的导出：

  • Plain Text
  • HTML
  • PDF
  • Microsoft Word

文档(File)

除了从本地建立文档之外，还可以链接DropBox和Google Drive

编辑视图(View)

  • 夜间模式(Night Mode)，晚上睡不着觉的话，这个功能是很好的

  • 焦点模式(Focus Mode)，当你编辑某一段文字时，编辑器会高亮当前段，实际上是将其他字段变灰。这样更容易集中注意力

  • 字数统计(Word Count)，字数统计会显示三种信息:

    • Characters，包含页面所有字符(包括换行符)
    • Words，所有语义字符(字母，数字，etc.)
    • Sentence，语段，每写一个段落，count++

预览(Preview)

码着码着，想看看实际效果，那么可以点顶栏最右边的”>”进行预览。

更简单的方法是在编辑界面向右滑动进入预览界面，再向左滑动可回到编辑界面。这个还真挺方便的。

预览模式下，有三种字体可选择

  • Mordern(Sans)
  • Classic(Serif)
  • Manuscript(Mono)

小结

总的来说，iA Writer可以算是移动端一款非常优秀的Markdown编辑器，界面友好，功能齐全。然而，基于移动设备的硬伤，弄一张图片上去真是不太方便，颇费周折
