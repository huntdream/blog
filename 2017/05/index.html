Linux常用命令

发表于 2017-05-02 | 分类于 Linux

文件操作(File Commands)

ls

列出当前目录下所有文件和目录（不包括隐藏文件和目录）

常用命令参数组合：

cd

进入某一目录

常用命令参数组合：

pwd

显示当前工作目录

mkdir

创建目录

rm

删除文件(rm大概是remove的缩写吧)

常用命令参数组合：

cp

复制文件

常用命令参数：

touch

创建或更新文件

###cat

得到解释

简单的说就是读取文件内容并输出

常用命令参数：

进程管理 (Process Management)

ps

显示当前活动的进程

top

显示所有正在运行的进程
